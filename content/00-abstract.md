This work presents Curve1024, a 1024-bit pairing-friendly elliptic curve ($E: y^2 = x^3 + 41$, embedding degree $k=18$) achieving 256-bit ECDLP security, constructed via a modified Cocks-Pinch algorithm that guarantees two-adicity of at least 36 in both the base field and scalar field primes — exceeding the industry-standard BLS12-381 curve. The construction constrains the cyclotomic seed so that the resulting primes are inherently compatible with NTT-based polynomial arithmetic used in modern proof systems. A multi-criteria readability scoring system further selects human-auditable primes from the valid candidate space without weakening security, enabling transparent parameter verification. The accompanying Rust implementation provides a reproducible test suite that verifies correct group arithmetic, validates pairing-based signature operations, and confirms resistance against MOV, Anomalous, and TNFS attacks — offering reviewers an independently executable proof of the curve's cryptographic soundness.
